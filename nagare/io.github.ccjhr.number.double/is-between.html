isBetween

infix inline fun <T : Double?> AssertionContext<T>.isBetween(boundaries: ClosedFloatingPointRange<Double>)

Verifies that the Double under test is between a lower and an upper boundary. To be successful the value must not be equal to the lower or upper boundary.
